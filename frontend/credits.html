Image Credits

Bird photographs on Sunbird are sourced from openly licensed collections. Each image is attributed to its photographer with the applicable license displayed alongside it.

Wikimedia Commons
Primary source. Images are retrieved via the MediaWiki API using the species' scientific name. Only images with open licenses (CC BY, CC BY-SA, CC0, or Public Domain) are used.
Licenses vary per image and are displayed individually.
iNaturalist
Fallback source when no suitable Wikimedia image is available. Only research-grade observations with CC0, CC BY, or CC BY-SA licensed photos are used.
Licenses vary per image and are displayed individually.

Bird observation data

Species sightings are provided by the eBird API (Cornell Lab of Ornithology), used under their API terms of use.

License requirements

Creative Commons licenses require attribution. Each bird image on Sunbird displays the photographer's name and the applicable license. If you believe an image is incorrectly attributed, please contact us.
